Example 1

Dobby cliff dived from 250 ft high cliff. What's the instanteous speed at t = 2 and average speed during first 3 seconds of the fall?

Solution

Assuming free fall under gravity with no initial velocity, it will fall y = 0.5 * g * t^2 in t sec from initial position.

julia> using Symbolics

julia> @variables t y g
3-element Vector{Symbolics.Num}:
 t
 y
 g

julia> D = Differential(t)
(::Symbolics.Differential) (generic function with 3 methods)

julia> y = 0.5 * g * t^2
0.5g*(t^2)

julia> yₜ = expand_derivatives(D(y)) # getting derivative
g*t

julia> speed_at_t2 = simplify(substitute(yₜ, Dict(g => 9.8, t=> 3))) # assume the free fall on earth
29.400000000000002

Average Speed will distance covered in the first 3 seconds of the fall, so 

julia> function average_speed1(y2, y1, t2, t1)
     Symbolics.value((y2 - y1) / 3)
end
average_speed1 (generic function with 1 method)

julia> yₜ₌₃ = substitute(y, Dict(g => 9.8, t => 3))
44.1

julia> yₜ₌₀ = substitute(y, Dict(g => 9.8, t => 0))
0.0

julia> averagespeed_till_t3 = average_speed1( yₜ₌₃, yₜ₌₀, 3, 0)
14.700000000000001
