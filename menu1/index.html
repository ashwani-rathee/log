Example 1

Dobby cliff dived from 250 m high cliff. What's the instanteous speed at t = 2 and average speed during first 3 seconds of the fall?

Solution

Assuming free fall under gravity with no initial velocity, it will fall y = 0.5 * g * t^2 in t sec from initial position.

julia> using Symbolics, Plots

julia> @variables t g # defining  the variables we intend to use in our function
2-element Vector{Symbolics.Num}:
 t
 g

julia> D = Differential(t) # dy/dt, this dt is Differential(t)
(::Symbolics.Differential) (generic function with 3 methods)

julia> y(g,t) = 0.5 * g * t^2 # motion equation
y (generic function with 9 methods)

julia> @register_symbolic y(g, t)

# gr()
# tvec = range(0, 5, length = 100)
# plot(y.(9.8, tvec))

// Image matching '/assets/menu1/code/symbol.svg' not found. //

So what's instantaneous speed at t = 2?

julia> yₜ = expand_derivatives(D(y)) # getting derivative
0

julia> speed_at_t2 = simplify(substitute(yₜ, Dict(g => 9.8, t=> 3))) # assume the free fall on earth
0

So Instantaneous speed at t=2 is 29.4 m/s.
n Average Speed will distance covered in the first 3 seconds of the fall, so 

julia> function average_speed1(y2, y1, t2, t1)
     Symbolics.value((y2 - y1) / 3)
end
average_speed1 (generic function with 1 method)

julia> yₜ₌₃ = y(9.8, 3)
44.1

julia> yₜ₌₀ = y(9.8, 0)
0.0

julia> averagespeed_till_t3 = average_speed1( yₜ₌₃, yₜ₌₀, 3, 0)
14.700000000000001

Average Speed is 14.7 m/s in first 3 seconds of the fall.
